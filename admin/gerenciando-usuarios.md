# Gerenciando Usuários

O sistema possui dois tipos de usuário que o administrador pode cadastrar:

- **Motoboy**: realiza as entregas
- **Cliente (Empresa)**: quem solicita as entregas

---

## Cadastrando um Motoboy

1. No menu lateral, clique em **Usuários**
2. Clique na aba **Motoboys**
3. Clique no botão **Novo Motoboy**
4. Preencha os dados:
   - Nome completo
   - Nome de usuário (será usado no login)
   - Senha
   - Telefone
   - Tipo de veículo (moto, bicicleta, carro)
5. Clique em **Salvar**

O motoboy já poderá acessar o sistema com o usuário e senha cadastrados.

---

## Cadastrando uma Empresa Cliente

1. No menu lateral, clique em **Usuários**
2. Clique na aba **Clientes**
3. Clique em **Novo Cliente**
4. Preencha:
   - Nome da empresa
   - Nome de usuário (login)
   - Senha
   - Telefone / E-mail
   - Endereço (ponto de partida padrão das coletas)
5. Clique em **Salvar**

---

## Editando um Usuário

1. Encontre o usuário na lista
2. Clique no ícone de lápis (editar)
3. Faça as alterações necessárias
4. Clique em **Salvar**

---

## Ativando e Desativando um Usuário

Para bloquear temporariamente o acesso de um motoboy ou cliente sem excluir o cadastro:

1. Localize o usuário na lista
2. Use o botão de toggle (ativar/desativar) ao lado do nome

Usuários desativados não conseguem fazer login.

---

## Grupos de Trabalho

Grupos de trabalho permitem organizar os motoboys por turno, região ou equipe.

1. No menu, clique em **Grupos de Trabalho**
2. Clique em **Novo Grupo**
3. Dê um nome ao grupo (ex: "Turno Manhã", "Zona Norte")
4. Adicione os motoboys desejados ao grupo
5. Clique em **Salvar**

Os grupos ajudam a filtrar e visualizar os motoboys no painel operacional.
